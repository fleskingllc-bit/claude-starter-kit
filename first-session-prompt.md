# 初回セッション用プロンプト

以下を Claude の最初のセッションでコピー＆ペーストしてください。
Claude が自動で初期設定を行います。

---

## コピー開始 ↓

```
ダウンロードフォルダにスターターキット（starter-kit）があります。
以下の手順で初期設定を行ってください。各ステップで何をするか説明してから実行してください。

1. ダウンロードフォルダでスターターキットを見つけてください（starter-kit フォルダまたは ZIP を解凍したフォルダ）

2. キット内のファイルを正しい場所に配置してください:
   - CLAUDE.md → ホームフォルダ直下（~/CLAUDE.md）
   - .claude/ フォルダ → ホームフォルダ直下（~/.claude/）
     ※ 既に .claude/ が存在する場合は、中身をマージしてください（既存ファイルは上書きしない）

3. ホームフォルダに配置した CLAUDE.md を読み込み、「基本情報」セクションの未入力項目を私に質問して埋めてください:
   - 会社名
   - 担当者（氏名）
   - 役職
   - 業種

4. PCのクラウドストレージを自動検出してください:
   - ~/Library/CloudStorage/ （Mac）または ~/OneDrive/ や ~/Google Drive/ を探す
   - Google Drive、OneDrive、その他のクラウドストレージのどれが使われているか特定する
   - 見つかったクラウドストレージ内に Obsidian_Vault フォルダがなければ作成する
   - CLAUDE.md の「ファイル管理」セクションのパスを、検出したクラウドストレージのパスに自動修正する
   - クラウドストレージが見つからない場合は、どのサービスを使っているか質問する

5. Obsidian Vault 内に以下のフォルダ構成を作成してください（既にあればスキップ）:
   - 00_inbox/
   - 10_projects/
   - 20_areas/
   - 30_resources/
   - 40_archive/
   - templates/

6. スターターキットの templates/ フォルダ内のテンプレートファイルを Obsidian Vault の templates/ にコピーしてください:
   - meeting-note.md（会議メモ用）
   - project-note.md（プロジェクトノート用）
   - daily-note.md（日次メモ用）

7. 設定内容を要約して報告してください
```
