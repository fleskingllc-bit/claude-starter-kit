# 作業ルール

- 日本語で回答すること
- 専門用語を使わず、わかりやすい言葉で説明すること
- ファイルを作成・変更する前に、必ず内容を説明して確認を取ること
- 重要なファイルを編集する前にバックアップを取ること
- エラーが出た場合は、何が起きたかを平易な言葉で説明すること

# ファイル管理

- ドキュメントの保存先: Google Drive 内の Obsidian Vault フォルダ
  - パス: ~/Library/CloudStorage/GoogleDrive-（メールアドレス）/マイドライブ/Obsidian_Vault/
- ファイル形式: Markdown (.md) を基本とする
- 作成したファイルには必ず YAML フロントマター（プロパティ）をつけること:

```yaml
---
title: ファイルのタイトル
date: 作成日（YYYY-MM-DD）
tags:
  - 関連タグ
category: カテゴリ名
status: draft / active / archived
---
```

# Obsidian Vault 構成

```
Obsidian_Vault/
├── 00_inbox/          ← 未整理・一時メモ
├── 10_projects/       ← プロジェクト別フォルダ
├── 20_areas/          ← 継続的な業務領域
├── 30_resources/      ← 参考資料・ナレッジ
├── 40_archive/        ← 完了・アーカイブ
└── templates/         ← テンプレート
```
